# Lakshya 5.0 Website

Official website for Lakshya 5.0, IIT Indore's flagship sports festival.

---

## Overview

Lakshya 5.0 is the annual sports festival of IIT Indore, bringing together athletes, competitors, and enthusiasts from institutions across the country.

This repository contains the development of the official Lakshya 5.0 website.

---

## Development Philosophy

The goal is not to build a traditional event website.

Lakshya 5.0 aims to deliver a cinematic, modern, and immersive web experience inspired by:

* Energy
* Motion
* Competition
* Legacy
* Fire
